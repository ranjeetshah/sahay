# 🌍 SAHAY

Help without money.

---

## 💡 What is this?
An open-source platform where people help each other through:
- 🧠 Advice
- 🎧 Listening
- 📚 Skill sharing

---

## 🚀 Vision
No money. No central control. Only human help.

- Database: PostgreSQL
- Frontend: Flutter
